# OCPI 3.0 (upstream-restricted)

OCPI 3.0 is under development in a **separate, access-restricted repository**
limited to EV Roaming Foundation contributors, and is not publicly available.

Because we cannot vendor the 3.0 text, this project treats 3.0 as a
**forward-scaffold** (milestone **M9**): version negotiation will recognise
`3.0`, and type stubs may be added, but the implementation logic is **deferred**
and tracked with the `blocked-upstream` label until the spec is public.

This follows the project's design philosophy: *defer logic, not schema.*

Upstream: https://github.com/ocpi/ocpi (see the README for the `ocpi-3` repo).
